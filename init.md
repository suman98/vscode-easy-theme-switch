Create a VS Code extension that enables users to manage and switch between multiple installed color themes, such as VS Code Dark, VS Code Light, Dracula, and more. Add a theme picker button to the bottom bar; when clicked, it opens a popover window displaying all available themes. The user can select a theme from this list, and the extension will immediately apply the chosen theme.

There should also be a settings button that allows users to customize their theme list by adding or removing themes they frequently switch between.

App flow:

- Display a theme picker button in the status bar.
- When clicked, show a list of the user's selected (favorite) themes.
- When a theme in the list is clicked, apply it immediately.
- Provide a settings button that lets users manage which themes appear in their personal theme library.
