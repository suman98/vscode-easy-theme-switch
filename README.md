# Theme Picker

Switch between your favorite VS Code color themes from the status bar.

## Features

- **Status bar icon** — one click opens your theme library
- **Live preview** — themes preview as you navigate the list
- **Escape to cancel** — reverts to original theme if you close without picking
- **Per-row remove** — trash icon on each theme removes it instantly
- **Add button** — `+` in the picker header adds new themes without leaving
- **Manage via command palette** — `Theme Picker: Manage Theme Library`

## Usage

1. Click the `◈` icon in the status bar (bottom right).
2. If your library is empty, choose **Add Themes** to populate it.
3. Arrow-key through themes — each one previews live.
4. Press **Enter** to apply, **Escape** to revert.

### Managing your library

**From inside the picker:**
- Click `+` (top right) to add more themes.
- Click the trash icon on any row to remove it.

**From the command palette:**
- `Cmd+Shift+P` → `Theme Picker: Manage Theme Library` → Add / Remove.

### Adding themes (multi-select)

Use **Space** to check multiple themes, then **Enter** to confirm. Only themes not already in your library are shown.

## Requirements

VS Code 1.75 or later.
